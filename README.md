## 📋 Sobre
 
Aplicação web simples para controle de estoque. Consome uma API REST via **MockAPI** para listar e cadastrar materiais.
 
## ✨ Funcionalidades
 
- 📦 Listar materiais via `GET`
- ➕ Cadastrar novos itens via `POST`
- 🔄 Atualização automática da lista após cadastro
## 🗂️ Estrutura
 
```
📁 almoxarifado/
 ├── index.html
 ├── style.css
 └── main.js
```
 
## 🔌 API
 
**Base URL:** `https://6a29eb54f59cb8f65f1dcaff.mockapi.io/Materiais`
 
| Método | Descrição |
|--------|-----------|
| `GET` | Lista todos os materiais |
| `POST` | Cadastra um novo material |
 
## 🚀 Como usar
 
```bash
git clone https://github.com/seu-usuario/almoxarifado.git
```
 
Abra o `index.html` no navegador ou use o **Live Server** no VS Code.
